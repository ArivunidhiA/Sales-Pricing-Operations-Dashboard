# Sales & Pricing Operations Dashboard

## Project Overview
This project creates a comprehensive sales and pricing operations dashboard using real retail sales data from the AdventureWorks sample database. The dashboard provides actionable insights into sales trends, pricing strategies, and operational efficiency.

## Features
- Automated data fetching from public datasets
- Interactive visualizations using Plotly
- Key business metrics and insights
- No manual input required - runs end-to-end automatically

## Visualizations
1. Monthly Sales Trend with Order Count
2. Sales Status Distribution
3. Product Price Distribution
4. Top Products by Price

## Requirements
- Python 3.7+
- Required packages listed in requirements.txt

## Installation & Usage

1. Clone this repository:
```bash
git clone https://github.com/yourusername/sales-dashboard.git
cd sales-dashboard
```

2. Install required packages:
```bash
pip install -r requirements.txt
```

3. Run the dashboard:
```bash
python main.py
```

4. Open the generated `sales_dashboard.html` file in your web browser to view the interactive dashboard.

## Output Files
- `sales_dashboard.html`: Interactive dashboard with visualizations
- `insights.txt`: Key business metrics and findings

## Data Source
This project uses the AdventureWorks sample database, specifically:
- Sales.SalesOrderHeader
- Production.Product
